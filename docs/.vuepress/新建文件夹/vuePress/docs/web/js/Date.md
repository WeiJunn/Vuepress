# Date

- js 中所有的日期相关的数据都通过 Date 来表示
- 如果直接创建 Date 对象,则会创建一个表示当前时间的对象
- 方法:

  ## getDay()

  - 获取当前日期对象是周几
  - 返回值是 0-6 0 表示周日

  ## getDate()

  - 获取当前日期

  ## getMonth()

  - 获取当前是几月
    - 返回值: 0-11 0 表示一月 显示具体月份要加一

  ## getFullYear()

  - 获取 4 位年份

  ## getTime()

  - 用来获取当前日期对象的时间戳
  - 时间戳是自 1970 年 1 月 1 日 0 时 0 分 0 秒,到现在时间所经过的毫秒数
  - 在计算机底层所有的时间都是以时间戳的形式存储

  ## Date.now()

  - 获取当前的时间戳
