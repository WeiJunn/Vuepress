# 字符串方法

  • 字符串本质上就是一个字符数组
    • 你在操作字符串时,完全可以将字符串想象为一个数组
    • 方法:

      • # concat()

        用来将两个或多个字符串连接起来

      • # charAt()

        根据索引获取字符串中的字符串

      • # charCodeAt()

        获取索引获取指定字符串的字符编码

      • # String.fromCharCode()

        根据编码返回字符

      • # indexof() lastindexof()

        查找一个字符中出现的索引的位置 last 是最后

      • # slice()

        用来才一个字符串中截取一个子串

      • # split()

        用来将一个字符串拆分为一个数组 参数: 需要一个分隔符作为参数会根据分隔符区拆分字符串
        let str = "a,b,c";
result = str.split(",");
console.log(result); // ["a","b","c"]

      • # toLowerCase() toUpperCase()

        转换大小写 lower 小 Upper 大

      • # trim()

        去除前后空格

      • # trimEnd()

        去除后面的空格

      • # trimStart()

        去除前面的空格

      • # endsWith()

        检查一个字符串是否以指定内容结尾

      • # startsWith()

        检查一个字符串是否以指定内容开始

      • # repeat(5)

        重复一个字符串五次
