# 遍历对象

众所周不知对象是有索引滴，所有我们遍历对象的话需要用到 for in

for(let  key in 对象){

}
for(let key in obj){
    console.log(key) 输出的是对象的属性名
    console.log(obj[key]) 输出的是对象的属性值
}
